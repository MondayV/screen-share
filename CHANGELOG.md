# 更新日志

## [1.4.2] - 2026-05-14

### 新增
- 五款皮肤系统：默认、深色、赛博朋克、手账涂鸦、像素比特，可在设置面板中切换
- 皮肤选择持久化存储，应用重启后保持选择
- 赛博朋克皮肤包含霓虹光效、扫描线、移动网格等动态特效
- 手账涂鸦皮肤采用暖色调手绘贴纸风格
- 像素比特皮肤实现 8-bit 街机像素美学

### 修复
- 修复皮肤切换后全局深色样式污染问题
- 提升手账涂鸦皮肤文字对比度，确保可读性
- 增大像素比特皮肤基础字号，改善中文可读性
- 修复像素比特皮肤设置界面缺失皮肤主题选项的问题
- 修复赛博朋克皮肤下拉框文字下半截遮挡问题
- 修复所有皮肤下拉框宽度不足以显示完整文字的问题

### 优化
- 清理开发过程中生成的临时风格指南文件
- 移除多余的 Claude Code 指令文件，保持仓库整洁
- 信令服务器内嵌至主进程，用户无需手动启动
