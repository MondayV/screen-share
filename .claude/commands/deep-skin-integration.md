# 指令：五款皮肤深度差异化集成

## 目标
将现有五款皮肤（默认、深色、极夜霓虹、手账涂鸦、像素比特）的差异化做到极致：  
- 待机主界面布局、装饰、动效完全不同  
- 所有交互控件（按钮、输入框、开关、状态灯、通知）拥有专属造型  
- 设置面板、关于页面各有风格化排版和图形元素  
- 彻底解决“输入框默认深色”问题，每个皮肤适配  
- 切换皮肤时视觉冲击明显，朋友一眼能认出当前皮肤

## 规则
- 使用 `[data-theme="xxx"]` 选择器为每个皮肤编写**独立 CSS 规则文件**（如 `cyberpunk.css` 内不仅定义变量，还要写具体组件样式）
- 不破坏功能，仅修改样式，必要时可为皮肤创建特定的 Svelte 条件渲染（如 Cyberpunk 的扫描线背景需要额外 DOM 节点）
- 所有中文提示保持
- 输入框必须逐皮肤设计：默认/深色采用传统边框，Cyberpunk 霓虹下划线，Doodle 横线纸，Pixel 像素凹槽
- 待机界面（连接前）需要有皮肤特征：Doodle 可展示手绘插画，Cyberpunk 有故障标题，Pixel 有街机高分榜，深色极简，默认原样
- 设置/关于页不只是换色，要重构布局：Cyberpunk 用等宽字体终端风格，Doodle 用手写体手账列表，Pixel 用像素菜单

## 执行步骤

### 1. 审查并列出所有需皮肤化的组件
- 待机主界面（`App.svelte` 或 `Home.svelte`）
- 连接/共享界面（`Host.svelte`, `Client.svelte`）
- 底部工具栏按钮（共享、麦克风、摄像头、录制等）
- 顶部栏（会议码、用户头像）
- 右侧面板（聊天、参会者列表）
- 输入框（短码输入、设置项、聊天输入）
- 开关/切换按钮（如批注开关、音频开关）
- 状态指示器（信令连接状态）
- 通知弹窗
- 设置面板、关于窗口
- 滚动条、选中文本样式

### 2. 重构主题文件结构
废弃原先仅含变量的单文件，新建 `src/styles/themes/` 下每个皮肤一个完整 CSS 文件，内部结构：
```css
[data-theme="cyberpunk"] {
  /* 1. 变量定义 */
  --bg-primary: ...;
  ...
  /* 2. 全局装饰（扫描线、背景网格） */
  body::before { ... }
  /* 3. 组件覆盖：按钮、输入框、开关等 */
  .btn-primary { ... }
  input[type="text"] { ... }
  .toggle { ... }
  /* 4. 特殊组件样式（待机界面、设置等） */
  .home-screen { ... }
  .settings-panel { ... }
}
