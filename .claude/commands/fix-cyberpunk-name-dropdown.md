# 指令：赛博朋克皮肤改名与下拉框文字遮挡修复

## 目标
1. 将皮肤“极夜霓虹”的中文名称和对应的标识全部改为“赛博朋克”，保持英文标识为 `cyberpunk` 不变（CSS 选择器、主题文件、存储 key 等）。
2. 修复赛博朋克皮肤下，皮肤主题下拉框文字下半部分被遮挡的问题，确保“赛博朋克”四个字完整显示。

## 修复要求

### 1. 全局更名
- 在所有 UI 组件中（设置面板、皮肤切换器、关于信息等），将用户可见的“极夜霓虹”文字替换为“赛博朋克”。
- 对应的数据属性值（如 `data-theme="cyberpunk"`）和 CSS 文件引用保持不变，以免破坏已写好的样式。
- 更新所有相关注释和文档字符串中的名称，但无需改动代码逻辑。

### 2. 下拉框文字遮挡修复
- 在 `cyberpunk.css` 中定位到下拉框（`select`）相关样式。
- 问题原因：可能 `line-height` 设置过小，或 `padding` 不足，导致文字垂直方向被裁剪；也可能是 `height` 固定且小于所需高度。
- 修复方法：
  - 增加 `line-height: 1.5` 或明确设置 `line-height: normal`。
  - 设置 `padding: 8px 12px`（上下给予足够空间）。
  - 移除固定的 `height` 限制，或设为 `height: auto`。
  - 确保 `overflow: visible` 或不设置 `overflow: hidden`。
  - 如使用 `Press Start 2P` 等像素字体，需增大 `font-size` 至 14px 并同步调整行高。
- 同时检查下拉箭头自定义样式是否占用过多空间，适当调整其定位。

### 3. 测试验收
- 切换到赛博朋克皮肤，进入设置面板。
- 确认皮肤名称显示为“赛博朋克”。
- 下拉框展开前后，“赛博朋克”四个字完整显示，无上半截或下半截缺失。
- 其他皮肤不受影响。
