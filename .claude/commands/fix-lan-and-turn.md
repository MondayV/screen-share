# 指令：局域网自动连接 + TURN 中继部署

## 目标
1. **局域网零配置**：主持方自动获取本机局域网 IP，UI 上显示完整信令地址（如 `ws://192.168.1.50:3456`），加入方可一键复制或扫码连接。
2. **防火墙友好**：应用启动时自动检查并提示 Windows 防火墙放行 3456 端口。
3. **跨网络兜底**：集成 TURN 服务器（提供自部署 coturn 脚本 + 临时免费 TURN 选项），确保不同 WiFi 下也能连接。
4. **向后兼容**：保留 `localhost` 连接能力（双实例测试仍可用）。

## 实施步骤

### 步骤一：信令服务器监听所有接口
- **文件**：主进程中的信令服务器代码（`src/main/signaling-server.js` 或主入口文件）。
- **修改**：将 WebSocket 服务器的监听地址从 `localhost` 改为 `0.0.0.0`，例如：
  ```js
  const wss = new WebSocketServer({ host: '0.0.0.0', port: 3456 });
